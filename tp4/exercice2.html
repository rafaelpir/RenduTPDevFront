Une liste de documents à télécharger

Mon retour sur cet exercice

Ce que j'ai appris

J'ai découvert les sélecteurs d'attributs CSS comme a[href$=".pdf"] pour cibler spécifiquement certains éléments selon leurs attributs.

Niveau de difficulté

⭐⭐⭐ (3/5) - Intermédiaire

C'était un bon challenge. Comprendre les sélecteurs d'attributs avec $= (se termine par), ^= (commence par) et *= (contient) demande un peu de pratique. Le plus dur était de bien organiser le CSS pour que chaque type de fichier ait sa propre icône sans dupliquer du code.

Mon avis sur l'intérêt pédagogique

Très utile pour styliser automatiquement des liens selon leur type sans ajouter de classes HTML.

Ce que je pense de mon résultat

Points positifs : Chaque type de fichier a sa propre icône et le bon drapeau s'affiche. Le CSS est bien structuré et facile à maintenir.
