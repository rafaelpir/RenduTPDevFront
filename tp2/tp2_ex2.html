Mon retour sur cet exercice

Ce que j'ai appris

J'ai découvert CSS Grid, un système de layout 2D avec display: grid, grid-template-columns/rows et grid-area pour placer précisément les éléments.

Niveau de difficulté

⭐⭐⭐⭐ (4/5) - Difficile

CSS Grid c'est clairement plus complexe que Flexbox au début. Comprendre la différence entre fr (fraction), % (pourcentage), et px (pixels) pour les colonnes, ça demande de la pratique. J'ai eu du mal avec grid-column-start et grid-column-end pour faire des items qui s'étalent sur plusieurs cases.

Mon avis sur l'intérêt pédagogique

CSS Grid est indispensable pour créer des layouts complexes, parfait pour les galeries, dashboards et layouts modernes.

Ce que je pense de mon résultat

Points positifs : La grille est bien structurée, les items occupent les bonnes zones
