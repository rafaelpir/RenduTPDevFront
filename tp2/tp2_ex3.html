Mon retour sur cet exercice

Ce que j'ai appris

J'ai pratiqué CSS Grid avec une galerie de 9 items en utilisant grid-template-columns/rows pour créer une grille régulière.

Niveau de difficulté

⭐⭐ (2/5) - Facile

Une fois qu'on a compris les bases de Grid dans l'exercice précédent, celui-ci est assez simple. C'est juste une application directe des propriétés de base. Le plus "difficile" c'était de gérer les espacements entre les items avec gap pour que tout soit bien aligné.

Mon avis sur l'intérêt pédagogique

Bon exercice de pratique, la répétition aide à bien ancrer les concepts de Grid.

Ce que je pense de mon résultat

Points positifs : La grille est bien alignée, les proportions sont bonnes, le code est propre et lisible
