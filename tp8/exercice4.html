Mon retour sur cet exercice

Ce que j'ai appris

J'ai découvert getImageData() pour lire les pixels d'un canvas et créer un color picker affichant RGB et HEX.

Niveau de difficulté

⭐⭐⭐⭐ (4/5) - Difficile

Comprendre comment les pixels sont stockés dans un canvas (tableau RGBA avec 4 valeurs par pixel) c'est pas évident. Il faut aussi gérer les événements de souris sur le canvas et calculer les coordonnées correctes. La conversion RGB vers hex demande de maîtriser les bases numériques. J'ai dû debugger pas mal pour que les coordonnées correspondent bien au pixel cliqué.

Mon avis sur l'intérêt pédagogique

Exercice fondamental pour comprendre comment les pixels sont stockés en mémoire sur canvas.

Ce que je pense de mon résultat

Points positifs : Le color picker fonctionne bien, les valeurs RGB et HEX sont correctes, l'interface est claire
