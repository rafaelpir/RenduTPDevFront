Mon retour sur cet exercice

Ce que j'ai appris

J'ai appris à créer un menu hamburger responsive en pur CSS avec :target et des ancres pour afficher/masquer le menu sans JavaScript.

Niveau de difficulté

⭐⭐⭐⭐ (4/5) - Difficile

Le menu hamburger c'est pas évident ! Comprendre la technique du :target demande de réfléchir différemment. Il faut aussi gérer le positionnement avec position: fixed, les z-index pour que le menu passe au-dessus du contenu, et les media queries pour que ça s'affiche que sur mobile. J'ai galéré avec les transitions pour que l'animation soit fluide.

Mon avis sur l'intérêt pédagogique

Excellent exercice pour comprendre la puissance du CSS et créer des interactions sans JavaScript, pattern très utilisé sur mobile.

Ce que je pense de mon résultat

Points positifs : Le menu fonctionne bien, l'animation est fluide, ça marche sur mobile et desktop
